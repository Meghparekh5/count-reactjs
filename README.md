Counter App (React)

A simple counter application built using React.js.
The counter value increases or decreases when the respective button is clicked.

Overview

This project demonstrates basic React concepts such as:

Functional Components

useState Hook

Event Handling

State Updates & Re-rendering

Tech Stack

React.js

JavaScript (ES6)

HTML5

CSS3

How It Works

The counter value is stored in React state.

Clicking Increase increments the value by 1.

Clicking Decrease decrements the value by 1.

React automatically re-renders the component when state changes.
